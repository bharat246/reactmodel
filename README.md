# Add to Cart Task Using Modal

This project is a ReactJS shopping interface built with Vite and Tailwind CSS. It fetches products from the Fake Store API, displays them in a responsive card layout, and lets users manage cart items inside a modal.

## Features

- Fetches products from `https://fakestoreapi.com/products`
- Displays product image, title, category, and price
- Adds items to cart from the product cards
- Shows an alert if the product is already in the cart
- Displays live cart count in the navbar
- Opens a modal with all cart items
- Removes items from the cart directly inside the modal
- Fully responsive layout for desktop and mobile

## Tech Stack

- ReactJS
- JavaScript
- HTML
- Tailwind CSS

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Open the local URL shown in the terminal.
